FSA SITE SETUP CHECKLIST v1.0
Maryland VITA Program - Free Self-Assisted Tax Preparation
Last Updated: October 2025
═══════════════════════════════════════════════════════════════════════════════

This checklist guides VITA coordinators in setting up Free Self-Assisted (FSA) 
tax preparation sites, including equipment, network configuration, physical 
space, and SIDN tracking.

SECTION 3: NETWORK & INTERNET SETUP
═══════════════════════════════════════════════════════════════════════════════

☐ Internet connectivity verified:
   - Download speed: _____ Mbps (min 10 Mbps per 3 users)
   - Upload speed: _____ Mbps (min 5 Mbps per 3 users)
   - Latency/ping: _____ ms (max 100ms recommended)

☐ Network security:
   ☐ Dedicated SSID for FSA workstations (not public WiFi)
   ☐ WPA3 or WPA2 encryption enabled
   ☐ Guest network isolated from main network
   ☐ Firewall configured for TaxSlayer access

☐ Required website access (whitelist if needed):
   ☐ taxslayerpro.com/*
   ☐ irs.gov/*
   ☐ marylandtaxes.gov/*
   ☐ Support video conferencing (Zoom, Teams, etc.)

☐ Device configuration:
   ☐ All software updates installed
   ☐ Antivirus/security software active
   ☐ Browser cache clearing procedures established
   ☐ Auto-save/password managers disabled on shared devices

☐ Backup connectivity:
   ☐ Mobile hotspot available with data plan
   ☐ Secondary internet provider as failover (optional)

SECTION 4: PHYSICAL SPACE CONFIGURATION
═══════════════════════════════════════════════════════════════════════════════

TAXPAYER WORKSTATION AREA
☐ Privacy partitions/dividers between stations
☐ Adequate lighting at each station
☐ Comfortable seating (adjustable chairs preferred)
☐ Desk space for documents and personal items
☐ Power outlets (2 per station minimum)
☐ Cable management to prevent trip hazards

☐ Accessibility compliance:
   ☐ Wheelchair-accessible stations (min 2)
   ☐ Adjustable-height desks/tables
   ☐ Screen readers available on request
   ☐ Large-print instructions posted
   ☐ ADA-compliant entrances and restrooms

NAVIGATOR SUPPORT AREA
☐ Semi-private area for phone/video consultations
☐ Soundproofing or noise reduction measures
☐ Whiteboard or display for training
☐ Secure storage for taxpayer documents (locked cabinet)

WAITING/RECEPTION AREA
☐ Check-in desk with greeter
☐ Waiting area seating (1 seat per 2 workstations)
☐ Sign-in sheet or digital kiosk
☐ Posted hours and services information
☐ Informational materials (FAQ, document checklist)

☐ Signage:
   ☐ Exterior building signage: "Free Tax Prep - VITA FSA"
   ☐ Directional signs from parking/entrance
   ☐ Room/workstation number labels
   ☐ Emergency exit signs visible
   ☐ Privacy notice posted

☐ Safety & security:
   ☐ First aid kit accessible
   ☐ Fire extinguisher present
   ☐ Security cameras (if applicable)
   ☐ Secure document shredding bin
   ☐ Hand sanitizer stations

SECTION 6: SIDN TRACKING SETUP

SIDN (Site Identification Number) is critical for tracking FSA usage and funding.

☐ Obtain Maryland FSA SIDN from IRS:
   Site SIDN: MARYLAND-VITA-2025-_________ (confirm with coordinator)

☐ Configure TaxSlayer Pro FSA link:
   Base URL: https://www.taxslayerpro.com/fsa
   Full tracking URL: https://www.taxslayerpro.com/fsa?sidn=MARYLAND-VITA-2025-_________

☐ SIDN usage tracking:
   ☐ Create QR codes with SIDN link for easy access
   ☐ Post SIDN link prominently at each workstation
   ☐ Include SIDN in all outreach materials
   ☐ Train navigators to share correct SIDN link

☐ Reporting procedures:
   ☐ Weekly SIDN usage reports to coordinator: ☐
   ☐ Track number of returns started: _____
   ☐ Track number of returns completed: _____
   ☐ Track navigator interventions: _____
   ☐ Monthly submission to IRS/state (deadline: 15th of month)

☐ Alternative SIDN tracking methods:

   ☐ Digital tracking spreadsheet: _______________
   ☐ Integrated with site management software: ☐
